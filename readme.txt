Redirect the author pages

== Description ==

Disable the author pages ( /author=? ) in wordpress and redirect to the user to another page.

If you love this plugin, <a href="https://paypal.me/arulprasadj?locale.x=en_GB">buy me a cup of coffee</a>.

== Screenshots ==

1. Settings menu
2. Settings Page

== Installation ==

1. Install the plugin from within the Dashboard or upload the directory `redirect-author-pages` and all its contents to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Configure it under settings Redirect Author Pages
